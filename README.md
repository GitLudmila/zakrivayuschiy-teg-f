# Закрывающий тег

HTML/CSS проект — дневник обучения вёрстке.

## Структура

- `index.html` — главная страница
- `styles/` — CSS-файлы (переменные, глобальные стили, анимации)
- `scripts/` — JavaScript
- `images/` — изображения
- `fonts/` — шрифты
- `svg/` — SVG-иконки

## Скрипты

```bash
npm install          # установка зависимостей
npm run check        # проверка форматирования (Prettier)
npm run format       # автоформатирование
npm run lint         # линтинг CSS (Stylelint)
npm run checkAll     # проверка форматирования + линтинг
npm run fixAll       # автоисправление форматирования + линтинга
```
